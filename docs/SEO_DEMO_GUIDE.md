## 🔧 故障排除

### 常见问题
1. **网络连接失败**
   - 检查URL是否正确
   - 确认网站是否可访问
   - 检查防火墙设置

2. **分析超时**
   - 网站响应过慢
   - 页面内容过大
   - 服务器负载过高

3. **结果异常**
   - 页面结构特殊
   - 动态内容较多
   - JavaScript渲染问题

### 解决方案
```
1. 使用备用测试URL
2. 调整超时设置
3. 使用本地文件分析
4. 检查服务器日志
```
