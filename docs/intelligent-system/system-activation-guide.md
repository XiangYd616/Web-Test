## 📋 快速启动检查清单

### 启动前准备
```yaml
环境检查:
  ✅ 系统要求:
    - Trae AI IDE 正常运行
    - MCP服务器连接正常
    - 网络连接稳定
    - 存储空间充足 (>1GB)
  
  ✅ 权限验证:
    - 文件读写权限
    - MCP工具访问权限
    - 网络访问权限
    - 系统配置权限
  
  ✅ 依赖检查:
    - 所有规则文件完整
    - MCP服务器可用
    - 记忆系统初始化
    - 推荐引擎就绪
```

### 启动步骤
```yaml
启动流程:
  步骤1_系统激活:
    操作: 输入 "启动超级大脑系统"
    预期: 系统组件逐一激活，状态显示正常
    验证: 所有模块显示 "✅ 就绪"
  
  步骤2_项目初始化:
    操作: 描述项目需求或目标
    预期: 系统自动识别项目类型和推荐阶段
    验证: 收到项目分析和阶段建议
  
  步骤3_阶段确认:
    操作: 确认或调整推荐的项目阶段
    预期: 进入指定阶段，显示任务和约束
    验证: 阶段目标清晰，任务列表完整
  
  步骤4_开始工作:
    操作: 按照阶段指导开始具体工作
    预期: 系统提供实时支持和智能建议
    验证: 推荐准确，工具使用高效
```
