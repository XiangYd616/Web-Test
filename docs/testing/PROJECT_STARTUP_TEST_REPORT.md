### 🔍 问题分析

#### **数据库连接问题**
- **配置**: PostgreSQL (localhost:5432)
- **数据库**: testweb_dev
- **用户**: postgres/postgres
- **状态**: ❌ **连接超时**

#### **可能原因**
1. **PostgreSQL服务未启动**
2. **数据库不存在**
3. **用户权限问题**
4. **防火墙阻止连接**

#### **解决方案**
```bash
# 1. 启动PostgreSQL服务
net start postgresql-x64-14

# 2. 创建数据库
createdb testweb_dev

# 3. 初始化数据库
npm run db:init

# 4. 重新启动后端
npm run dev
```
