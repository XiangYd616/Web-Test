### 技术相关
- **前端技术栈**: React 18 + TypeScript + Tailwind CSS
- **后端技术栈**: Node.js + Express + PostgreSQL
- **测试引擎**: K6 + Playwright + 自定义引擎
- **桌面版**: Electron + Electron Builder

### 功能模块
- **测试功能**: 压力测试、兼容性测试、安全测试、API测试
- **数据管理**: 测试历史、数据导出、实时监控
- **用户系统**: 认证登录、权限管理、主题切换
- **系统管理**: 配置管理、日志监控、数据备份
